Card\_Match


About Game:
- Load Game will load last played game (if game is not over) in the same state
- At start of a new game, the cards will be revealed of a few seconds. At this time, the cards will not be interactable


About Project:
- Used Sprites instead of Image/Button for the cards cause sprites are more efficient and flexible. Moreover any changes in UI element will set Canvas dirty, which is not scalable
- The cards are scaled to fit the target display area at the game start
